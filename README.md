**Installation**
1 - Installer composer sur : https://getcomposer.org/download/
2 - Ouvrir un serveur web Apache/PHP sur XAMPP ou avec la commande 
`php -S localhost:8888 -t mini-projet_galerie/`
3 - Ajoutez /web/index.php à l'URL
