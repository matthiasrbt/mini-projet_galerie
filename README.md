**Installation**
- Installer composer sur : https://getcomposer.org/download/
- Ouvrir un serveur web Apache/PHP sur XAMPP ou avec la commande 
`php -S localhost:8888 -t mini-projet_galerie/`
- Ajoutez /web/index.php à l'URL
